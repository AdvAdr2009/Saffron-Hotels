Our Origins

The hotel was started by the current owners, Advaiy Vikas and Adrith Vikas' father, Vikas KC. The first hotel was built in Kovalam, Thiruvananthapuram in 2001. Since then, 23 hotels have been built in various places across the country and 4 have been built abroad.

This website is an overview of the first hotel in Kovalam which has been upgraded multiple times.
